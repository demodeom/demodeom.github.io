# Padding

| 属性    | 描述       |     |
| ------- | ---------- | --- |
| padding | 设置内边距 |     |
| child   | 设置子元素 |     |
|         |            |     |

```dart
 Padding(
    padding: EdgeInsets.all(10),
    child: Text("Hello Flutter"),
)
```
