Flameshot 是一个功能强大的开源截图工具，专为 Linux 用户设计。它提供直观的界面和多种截图编辑功能，如注释、模糊处理和高亮显示。用户可以自定义快捷键、调整界面风格，并通过命令行进行自动化操作。Flameshot 支持多种图像格式，适合开发者、设计师和普通用户使用。

<!-- more -->

```bash
flatpak install flathub org.flameshot.Flameshot
```
