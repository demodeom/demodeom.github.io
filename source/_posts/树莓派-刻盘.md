<!-- more -->

![https://s1.ax1x.com/2022/06/12/X2c4rn.jpg](https://s1.ax1x.com/2022/06/12/X2c4rn.jpg)

## 下载刻盘工具

Raspberry Pi Imager 是树莓派社区提供的刻盘工具， 支持主流的操作系统， 支持所有型号的树莓派

[Raspberry Pi Imager 下载地址](https://www.raspberrypi.com/software/)


**Setp 1** 下载之后得到 `imager_1.7.2.exe`

**Setp 2** 双击 `imager_1.7.2.exe`

**Setp 3** 点击 `Install` 按钮

![X26w7V.png](https://s1.ax1x.com/2022/06/12/X26w7V.png)

**Setp 4** 勾选 `Run Raspberry Pi Imager`, 点击 `Finish` 按钮, 即可启动 `Raspberry Pi Imager`

![X26BkT.png](https://s1.ax1x.com/2022/06/12/X26BkT.png)

![X26IhD.png](https://s1.ax1x.com/2022/06/12/X26IhD.png)

## 选择操作系统

点击 `选择操作系统`

![X2rIhQ.png](https://s1.ax1x.com/2022/06/12/X2rIhQ.png)

点击 `Other general-purpose OS`

![X2r71s.png](https://s1.ax1x.com/2022/06/12/X2r71s.png)

点击 `Ubuntu`

![X2r5tg.png](https://s1.ax1x.com/2022/06/12/X2r5tg.png)

点击 `Ubuntu Server 22.04 LTS`

![X2rTpj.png](https://s1.ax1x.com/2022/06/12/X2rTpj.png)

## 选择 SD 卡

点击 选择 SD 卡

点击 SD 卡

## 刻录

点击 `刻录` （刻录会删除SD卡的内容）

![X2rvAU.png](https://s1.ax1x.com/2022/06/12/X2rvAU.png)

点击 `是`

![X2rxNF.png](https://s1.ax1x.com/2022/06/12/X2rxNF.png)

刻录过程耐心等待

![X2rzh4.png](https://s1.ax1x.com/2022/06/12/X2rzh4.png)

看到以下提示， 表示刻录成功， 点击 `继续`

![X2sp9J.png](https://s1.ax1x.com/2022/06/12/X2sp9J.png)

关闭 `Raspberry Pi Imager`

![X2s939.png](https://s1.ax1x.com/2022/06/12/X2s939.png)

## 开机

1. 树莓派插入 SD 卡
2. 接通电源（通电后会自动开机）
