Step1: 下载最新的源码, 解压

Step2: 切换到源码目录, 执行 phpize

Step3: 切换到源码目录, 执行 ./configure

Step4: 切换到源码目录, 执行 make

Step5: 切换到源码目录, 执行 sudo make install

Step6: 修改 php.ini 文件, 文件尾部追加 extension=swoole
