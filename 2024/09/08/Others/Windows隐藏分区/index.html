Windows隐藏分区

Win 隐藏分区

第一步: 打开 diskpart

打开 CMD, 执行 diskpart 命令

第二步: 列出磁盘列表

执行命令 list disk

第三步: 选择磁盘

选择分 EFI 分区所在的磁盘, 比如: select disk 0

第四步: 列出分区列表

执行命令 list volume

第五步: 删除挂载点

以 Z 盘符为例; 执行命令 select volume Z 选中分区; 执行命令 remove letter=Z 删除挂载点

第六步: 关闭cmd

执行命令 exit
