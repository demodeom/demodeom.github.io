# 说明文档

## 主题

[ fluid-dev /hexo-theme-fluid](https://github.com/fluid-dev/hexo-theme-fluid)


```
npm install --save hexo-theme-fluid
```

修改 Hexo 博客目录中的 _config.yml：

```
theme: fluid  # 指定主题

language: zh-CN  # 指定语言，会影响主题显示的语言，按需修改
```
